# Acceptable Use

Chopsticks is an orchestration platform that can operate at scale. If you run it, you are responsible for how it is used.

## Not Allowed
- Harassment, hate, threats, or targeted abuse.
- Spam, raids, or attempts to evade Discord enforcement.
- Token farming, credential theft, or user deception.
- Operating the bot in ways that violate Discord Terms of Service.
- Using pooled agents to disrupt other servers.

## Required Practices
- Configure permissions correctly and keep privileged commands restricted.
- Enable rate limits and do not disable safety checks.
- Keep tokens/secrets out of repos and logs.
- Patch and update regularly.

## Enforcement
This policy is a project guideline and does not replace Discord policies or the LICENSE terms.
