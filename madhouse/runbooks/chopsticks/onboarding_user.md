# User Quick-Start Guide

## 1. Commands & Help
- **Discover:** Run `/help` for a full command list.
- **Specifics:** Use `/help category:<name>` or `/help command:<name>`.
- **Tutorials:** Run `/tutorials` for interactive walkthroughs.

## 2. Economy
1. **Claim Daily:** `/daily` (Resets every 24h).
2. **Check Stats:** `/balance` and `/streak`.
3. **Play:** `/casino slots bet:<amount>`.

## 3. Music
1. **Join Voice:** Enter any voice channel.
2. **Play:** `/music play query:<song|url>`.
3. **Control:**
   - `/music queue`: View upcoming tracks.
   - `/music skip`: Skip current.
   - `/music stop`: Clear and leave.
   - `/music save name:<name>`: Save current queue.

## 4. AI Features
- **Translate:** `/ai translate text:<text> target_language:<lang>`.
- **Summarize:** `/ai summarize count:<number>` (Requires mod permissions).
- **Personal Key:** `/ai key set key:<token>` (If enabled by admin).

## 5. FAQ
**Q: Command fails?**
A: Check your Discord permissions or server roles.

**Q: Daily already claimed?**
A: Resets 24h after your last claim.

**Q: Bot left voice?**
A: Happens after 5 minutes of inactivity.

**Q: Trade expired?**
A: Trades time out after 5 minutes. Items are not lost.
